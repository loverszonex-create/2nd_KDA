# 키우Me 챗봇 데모 가이드

## 🎯 Mock AI 챗봇 체험 가이드

이 애플리케이션에는 실제 LLM처럼 작동하는 Mock AI 챗봇이 구현되어 있습니다!

## 🚀 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. 브라우저에서 http://localhost:3000 접속
```

## 💬 챗봇 테스트 방법

### 1단계: 홈페이지에서 종목 선택
- 홈페이지에서 "삼성전자", "SK하이닉스", "금융주 팀톡" 중 하나를 클릭하세요.
- 또는 검색창에서 직접 검색할 수 있습니다.

### 2단계: 추천 질문 클릭
채팅 페이지에서 다음과 같은 추천 질문 버튼이 표시됩니다:
- **"키움증권 의견을 알려 줘"**
- **"AI 버블론은 뭐였는데?"**

이 버튼들을 클릭하면 AI가 자동으로 답변합니다!

### 3단계: 연속 대화
AI 답변 후에 새로운 추천 질문이 나타납니다. 계속 클릭하면서 대화를 이어가세요:

```
사용자: "AI 버블론은 뭐였는데?"
  ↓
AI: [AI 버블론에 대한 상세 설명]
  ↓
새 추천: "삼성전자는 언제 다시 오를까?" / "지금이 매수 타이밍일까?"
  ↓
사용자: "삼성전자는 언제 다시 오를까?"
  ↓
AI: [주가 전망 분석]
  ↓
새 추천: "분할 매수 전략을 추천해줘" / "목표가를 어디로 잡을까?"
```

### 4단계: 직접 입력
하단의 입력창에 직접 질문을 입력할 수도 있습니다. AI가 키워드를 분석해서 적절한 답변을 제공합니다.

## 🔌 실제 LLM 연동 준비

Mock 시스템은 실제 LLM API로 쉽게 교체할 수 있도록 설계되었습니다.

### 1. API 키 설정

`.env` 파일 생성:
```bash
VITE_OPENAI_API_KEY=sk-...
VITE_ANTHROPIC_API_KEY=sk-ant-...
```

### 2. API 함수 교체

`src/utils/chatAPI.js`의 `getAIResponse` 함수를 실제 API 호출로 교체:

```javascript
export async function getAIResponse(userMessage, stockName) {
  const response = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${import.meta.env.VITE_OPENAI_API_KEY}`
    },
    body: JSON.stringify({
      model: 'gpt-4',
      messages: [
        {
          role: 'system',
          content: `당신은 "${stockName}"에 대한 전문 투자 상담 AI입니다.`
        },
        {
          role: 'user',
          content: userMessage
        }
      ]
    })
  })
  
  const data = await response.json()
  
  return {
    content: parseResponse(data.choices[0].message.content),
    suggestions: extractSuggestions(data.choices[0].message.content)
  }
}
```
